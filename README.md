## Overview

A clean, beginner-friendly Jupyter notebook demonstrating **how to create pandas DataFrames from Python dictionaries** — one of the most essential skills for anyone starting in data analysis.

## What You'll Learn

- Creating `pd.DataFrame()` from dictionaries
- Handling different data structures and column names
- Real-world style datasets (Employees, Students, HR records)
- Basic DataFrame inspection and dictionary operations

## Notebooks

- `task_2.ipynb` — Main tutorial with 3 complete examples

## Datasets Included

1. **Simple Employee Data** — Name, Age, Course, Performance
2. **University Student Results** — Roll No, CGPA, Attendance, etc.
3. **Detailed Employee Records** — 10 employees with salary, department, experience, performance score, and more
